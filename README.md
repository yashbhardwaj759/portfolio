## How to Use

1. Open `index.html` in any web browser
2. That's it! No installation needed.

## Deploy to Get HTTPS Link

### Option 1: GitHub Pages (Free & Easy)

1. Create a GitHub account
2. Create a new repository named `portfolio`
3. Upload all files (index.html, style.css, script.js)
4. Go to Settings → Pages
5. Select main branch and save
6. Your site will be live at: `https://yourusername.github.io/portfolio`

### Option 2: Netlify (Free)

1. Go to https://netlify.com
2. Sign up for free
3. Drag and drop your folder
4. Get instant HTTPS link!

### Option 3: Vercel (Free)

1. Go to https://vercel.com
2. Sign up
3. Upload your folder
4. Get HTTPS link instantly
